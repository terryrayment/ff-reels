# F&F Sales — Tracker & Playbook

Working sales tracker for Friends & Family commercial production. Lives in the repo so it's versioned, searchable, and survives laptop changes.

## Files

| File | Purpose |
|------|---------|
| [`SALES.md`](./SALES.md) | **The master tracker.** 60-prospect status table, strategy summary, rep relationships, cadence. Edit this daily. |
| [`INTROS.md`](./INTROS.md) | All 60 founder intro emails. Reference doc; rarely edited. |
| [`STRATEGY.md`](./STRATEGY.md) | West Coast strategy with CCCo. + Gunder, the four wedges, target accounts. |
| [`REPS.md`](./REPS.md) | Rep relationships, economics, what works/doesn't. |

## How to use

1. Open [`SALES.md`](./SALES.md).
2. Update the **Status** column as you send, get replies, book calls.
3. Each prospect row links to its intro email in [`INTROS.md`](./INTROS.md).
4. Weekly review: scan for ⬜ → 📤 transitions; chase 📤 over 7 days old with the follow-up rule.

## Status legend

- ⬜ Not Sent
- 📤 Sent
- ⏳ Follow-up Sent
- 💬 Replied
- 📞 Call Booked
- 🎬 In Bid
- 🏆 Won
- ❄️ Cold (no reply after follow-up)
- ❌ Pass

## Quick rules

- **Don't send all 60 in one week.** ~6/week over 8–10 weeks.
- **Verify ⚠️ emails on Hunter.io** before sending.
- **Reply within 4 hours** when a founder responds.
- **One follow-up at 8 days.** Stop after that unless trigger event.
- **Never CC the rep on cold-open emails.** Loop them in after the founder replies.
